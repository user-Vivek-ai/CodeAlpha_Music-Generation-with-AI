# 🎬 Tollywood AI Music Generator

An AI-powered music generator that creates Telugu film-style melodies using Indian raagas and mood-based composition. Built with HTML, JavaScript, Tone.js, and the Web Audio API.

# Live Demo Link : https://user-vivek-ai.github.io/CodeAlpha_Music-Generation-with-AI/

## ✨ Features

* AI-generated melodies
* 8 Tollywood moods
* 6 Indian raagas
* 5 virtual instruments
* Real-time playback
* Waveform visualizer
* MIDI & WAV export
* Single-file application

## 🎭 Available Moods

* Love Scene
* Action
* Sad / Emotional
* Celebration
* Devotional
* Comedy
* Suspense
* Folk / Janapada

## 🎵 Supported Raagas

* Shankarabharanam
* Kalyani
* Kharaharapriya
* Bhairavi
* Hamsadhwani
* Mohanam

## 🚀 How to Run

1. Download or clone this repository.
2. Open `tollywood_music_generator.html` in any modern browser.
3. Select a mood and raaga.
4. Click **Generate Music**.
